# Tests

1. Create `bootstrap_yii2_tests` database and run

   ```
   composer test-prepare
   ```

2. Run build tests

   ```
   composer test-build
   ```

3. Run tests

   ```
   composer test
   ```

4. Run tests with coverage

   ```
   composer test-coverage
   ```

5. Show coverage dashboard

   ```
   composer test-coverage-open
   ```
