# Tests

## Preparation

1. Create `bootstrap_yii2_tests` database and run
   ```
   composer test-prepare
   ```

2. Run build tests
   ```
   composer test-build
   ```

## Commands

- Run tests
  ```
  composer test
  ```

- Run tests with coverage
  ```
  composer test-coverage
  ```

- Show coverage dashboard
  ```
  composer test-coverage-open
  ```
